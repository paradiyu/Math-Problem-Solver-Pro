# 🎓 考研数学做题区助手 (Math Problem Solver Pro)

**考研数学做题区助手** 是一款基于 Google Gemini 3.1 & 2.0 大模型开发的自动化数学题目解析工具。它能够将你的 PDF 练习册转化为高颜值的“全手写”风格笔记，题目与解答过程完美融合，风格统一，适配手机阅读，助你随时随地复盘错题。

---

## 🚀 核心功能

* **智能图像净化**：自动识别题目区域，去除原始图片中的杂质、手指及背景噪点。
* **严谨推理过程**：调用 `gemini-3.1-pro` 推理引擎，确保数学逻辑、符号、公式严格准确。
* **统一手写风格**：采用“米白色横格纸 + 黑色中性笔 + 红色重点标注”的固定视觉风格。
* **手机适配排版**：生成垂直长图，完美适配移动端竖屏查看。

---

## 🛠️ 环境准备

### 1. 配置 API Key

本工具推荐使用 **神马中转 (WhatAI)** 接口以获得更稳定的多模态支持。

* 前往 [神马中转官网](https://api.whatai.cc) 注册并获取 `API_KEY`。
* 在脚本中修改以下配置：
```python
client = genai.Client(
    http_options=types.HttpOptions(base_url="https://api.whatai.cc"),
    api_key="你的_SK_KEY"
)

```



### 2. 安装依赖

确保你的 Python 环境已安装以下库：

```bash
pip install google-genai pillow

```

## 📂 操作流程

### 第一步：准备题目本

从互联网下载考研数学做题本（推荐：张宇、汤家凤、李永乐等老师的 660 题或 880 题）。

> **最佳实践**：为了保证识别精度，建议使用 **一页只有一道题** 的 PDF 版本。

### 第二步：PDF 转图片

使用 `pdftopng` 或 `pdf2image` 工具将 PDF 的每一页转换为独立的 PNG 图片。

```bash
# 使用 Linux/Mac 终端工具示例
pdftopng input_math.pdf problem_folder/lilin

```

转换后，你的文件夹应如下所示：

* `lilin-000001.png`
* `lilin-000002.png`
* ...

### 第三步：批量运行脚本

将脚本指向题目文件夹，程序将自动执行以下流程：

1. **文本识别**：分析题目内容。
2. **严谨解答**：生成标准的数学推导 Markdown。
3. **视觉渲染**：调用 Nano Banana 模型生成手写长图。

## ⚠️ 注意事项

* **图片大小**：脚本内置了自动压缩功能，建议原始 PNG 不要超过 10MB。
* **超长题目**：对于证明题等篇幅极长的题目，建议适当调高 API 超时时间（`timeout`）。
* **隐私安全**：请勿将带有明文 `API_KEY` 的代码上传至公开仓库，建议使用 `.env` 文件管理密钥。
